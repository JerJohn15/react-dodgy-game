## Dodgy Game

It's a game built in React ¯\\_(ツ)_/¯

[Check it out.](http://brandonstilson.com/dodgygame)

##Start Game 

To start the game on your local host:

```
git clone https://github.com/bbstilson/react-dodgy-game.git
cd react-dodgy-game
sudo npm i
sudo npm start
open http://localhost:8080
```
